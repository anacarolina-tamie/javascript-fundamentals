## 📚 Sobre o repositório
Este repositório contém os exercícios práticos do curso de **JavaScript Tutorial Full Course - Beginner to Pro** do canal do YouTube SuperSimpleDev. 


## 🎓 Sobre o curso
O curso **JavaScript Tutorial Full Course - Beginner to Pro** aborda uma jornada completa de desenvolvimento web, começando pelos conceitos mais básicos de JavaScript e evoluindo gradualmente para tópicos avançados. Ao longo das aulas, há uma forte integração entre **HTML, CSS e JavaScript**, além de introduções práticas a:
* **Bibliotecas externas** e gerenciamento de código com **Módulos**
* **Manipulação de Document Object Model (DOM)**
* **Programação Orientada a Objetos (POO)**
* **Noções de backend**, requisições assíncronas, callbacks e *async/await*

O curso conta com um projeto prático de **desenvolvimento de um e-commerce**, replicando a interface e as funcionalidades principais do fluxo de compra da Amazon, unindo todo o conhecimento prático adquirido.


## 🎯 Objetivo
O propósito deste repositório é aprender e praticar os fundamentos do JavaScript, praticando a lógica de programação, manipulação do DOM e a criação de interfaces web interativas do zero, construindo uma base sólida para o desenvolvimento front-end.


## 🛠️ Tecnologias
* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**
