## 📚 Sobre o projeto
Este repositório contém os exercícios práticos do curso de "JavaScript Tutorial Full Course - Beginner to Pro" do canal do YouTube SuperSimpleDev. 

Alguns dos exercícios e projetos práticos são baseados e utilizam os materiais do site [Super Simple Dev - Amazon Project](https://supersimple.dev/projects/amazon/), criado pelo instrutor do curso.
